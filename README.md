# Interior Design Website

A modern, responsive website for an interior design startup, built with React, Tailwind CSS, and Framer Motion for animations. The site features a glassmorphism effect, a responsive navigation bar with a hamburger menu, and multiple sections showcasing the startup's services.

## Technologies Used

- **React 18+**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Framer Motion**: A library for creating animations in React.
